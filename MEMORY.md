# Автопамять проекта AgroNDVI

## Ключевые факты
- **Владелец:** Валентюк Евгений Григорьевич (ВЕГ). Системный/бизнес-аналитик, после MiniProctor продолжает переход в ML.
- **Тип проекта:** пет-проект для портфолио, второй по счёту.
- **Домен:** агро + спутниковый мониторинг полей, прогноз продуктивности.
- **Срок:** плотные 7-10 дней.
- **Стек:** Python + rasterio + geopandas + LightGBM + Streamlit + folium. Без нейросетей в первой версии.
- **Опыт CV/ML** -- средний (один пет-проект MiniProctor за плечами). Геоданные -- новый стек, осваиваем в процессе.

## Цели
1. Довести проект до состояния «работающий pipeline + интерактивный UI + метрики на исторических данных».
2. Получить опыт работы со спутниковыми данными (Sentinel-2), геопространственными библиотеками (rasterio, geopandas) и time-series ML (LightGBM с лагами).
3. Показать проект на собеседованиях в банковский агро-сегмент: РСХБ-Цифровые решения в первую очередь, далее Сбер для агробизнеса, Совкомбанк, агрохолдинги, агростраховые.

## Целевые работодатели
- **РСХБ-Цифровые решения** -- основной таргет. Делают агро-скоринг через спутник, прогноз урожайности, риск-менеджмент.
- **Сбер для агробизнеса** -- agroindex.ru, отдельная команда внутри Сбера.
- **Совкомбанк Страхование** -- агростраховой риск-менеджмент через спутник.
- **Ингосстрах, СОГАЗ** -- агростраховые продукты, спутниковый мониторинг ущерба.
- **Агрохолдинги:** ЭФКО, Русагро, Содружество, Мираторг, Степь, Прогресс Агро.
- **Геотех-стартапы:** ExactFarming, OneSoil, Геомир -- работают в этой нише.

## Что НЕ надо делать
- Не добавлять нейросети в первую версию -- классический ML (LightGBM) + математика NDVI достаточны для MVP. U-Net для сегментации полей -- возможный второй этап.
- Не пытаться сделать прод-сервис -- задача показать процесс и метрики.
- Не закапываться в борьбу за лучший MAPE -- 15-20% уже хорошо для прогноза урожайности.
- Не тянуть платные сервисы (Sentinel Hub, Planet) -- всё на бесплатных Sentinel-2 через AWS / Copernicus.

## Решения по конфигурации (закрыты 2026-05-25 перед стартом Дня 1)
- **Регион:** Краснодарский край. Богатая статистика Росстата, хорошее покрытие Sentinel-2, самый «показательный» регион для агро-портфолио.
- **Культура:** озимая пшеница. Длинный вегетационный сезон (октябрь-июль) -- много NDVI-точек на time-series, массовая культура с большим объёмом исторических данных.
- **Границы полей:** рисуем 5-10 полигонов вручную в QGIS / geojson.io. Контроль качества 100%, независимость от чужих датасетов. EuroCrops для России беден.
- **Артефакты обучения:** локально в `models/` под `.gitignore`. Финальные веса -- в GitHub Releases как ассет к v1.0.0 (как делал MiniProctor).
